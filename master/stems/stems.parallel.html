stems.parallel.iter_chunks(dim_sizes, chunk_sizes=None)[source]¶

Return slices that help iterate over dimensions in chunks/blocks

dim_sizes : dict[str, int]
Dimension sizes
chunk_sizes : None, int, or dict[str, int]
Sizes of chunks. If int, will use the same size for all dimensions. If None, will iter over each element in all dimensions (e.g., one pixel)
dict[str, slice or int]
Mapping of dimension name to select statement (e.g., {'x': slice(0, 50), 'y': slice(0, 50)})

Decorator that maps a function across pixels and concatenates the result

Core dimensions to exclude from broadcast/selection. For example, if applying a time series model over all x/y in a DataArray, the core_dim could be ‘time’.

Index(es) of *args to pick as the data to iterate over.

Axis along which the arrays will be joined (passed to concat_func)

Function used to concatenate the data. Should take the axis keyword
